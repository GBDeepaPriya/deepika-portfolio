🌡️ Climate Trend Analyzer

Climate Trend Analyzer | Link

● Developed an end-to-end Climate Trend Analyzer to analyze long-term environmental patterns using time-series data such as temperature, rainfall, CO₂ levels, humidity, and sea-level trends.

● Implemented trend analysis using moving averages and performed anomaly detection using Z-score techniques to identify abnormal climate events.

● Built a forecasting system using ARIMA to predict future climate conditions and visualize long-term climate behavior.

● Designed an interactive Streamlit dashboard with dynamic filtering, parameter selection, seasonal analysis, and real-time visualization using Plotly.

Key Features:

    📈 Climate Trend Visualization

    🚨 Anomaly Detection System

    🔮 Future Climate Forecasting

    📅 Date Range Filtering

    🌦️ Seasonal Analysis

    📊 Multi-Parameter Comparison

    📥 Forecast CSV Download Option

Technologies:Python, Pandas, NumPy, Plotly, Streamlit, Statsmodels (ARIMA), Matplotlib, Scikit-learn
