🚆 Railway Seat Availability System

Railway Seat Availability System

● Built a web-based application to check train seat availability with a responsive user interface and backend database integration. The app also includes a speech synthesis feature that audibly reads seat availability to enhance accessibility.

Key Features:

  • Train seat availability based on user-selected train, date, and class
  • Speech synthesis output to assist visually impaired users
  • Backend database integration to simulate seat status
  • Seat Availability display based on categories (e.g., AC, Sleeper, General)
  • Validation for form inputs (train number, date, quota)

    • Tech Stack: HTML, CSS, JavaScript, PHP , MySQL
